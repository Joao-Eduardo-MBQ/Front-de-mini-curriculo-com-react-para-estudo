O objetivo principal deste projeto foi o aprendizado prático de HTML, CSS e React, além da configuração do ambiente com Vite.
Por isso, você encontrará muitos comentários detalhados no código — isso é intencional para facilitar o entendimento e servir como material de estudo.
O código não está otimizado para produção, mas sim estruturado para que alguém que esteja começando consiga acompanhar a lógica e o funcionamento.
